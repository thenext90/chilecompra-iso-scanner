# ChileCompra ISO Scanner 🎯

Buscador de oportunidades ISO en ChileCompra.
Estrategia de ventas de certificaciones para clientes CMS Consultores.

## Objetivo
Detectar licitaciones públicas que requieran certificaciones ISO
(9001, 14001, 45001) para usarlas como argumento de venta.

## Cliente piloto
**Mago Chic S.A. (Magochic)** — Aseo industrial

## Estructura
- `src/` — Código fuente
- `data/` — Datos descargados
